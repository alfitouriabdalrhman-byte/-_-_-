مراجعة سريعة – الكيمياء العضوية

إعداد الطالب: عبدالرحمن الفيتوري

الشيت 2 – Aromatic Hydrocarbons

التفاعلات الأساسية للبنزين (Electrophilic Substitution)
  • Halogenation → Benzene + X₂ / FeX₃ → Halobenzene
  • Nitration → HNO₃ / H₂SO₄ → Nitrobenzene
  • Sulfonation → SO₃ / H₂SO₄ → Benzenesulfonic acid
  • Friedel–Crafts Alkylation → R–Cl / AlCl₃ → Alkylbenzene
  • Friedel–Crafts Acylation → RCOCl / AlCl₃ → Acylbenzene
تفاعلات السلسلة الجانبية
  • Oxidation of Alkyl Side Chain → KMnO₄ (hot) → –COOH (دائمًا)
⚠️ يتكرر في الامتحان: نوع التفاعل + الشرط + الناتج النهائي فقط.

الشيت 3 – Alcohols, Phenols & Ethers

طرق تحضير الكحولات
  • Hydration of Alkenes → H⁺ / Markovnikov → Alcohol
  • Reduction of Aldehydes → LiAlH₄ / NaBH₄ → Primary Alcohol
  • Reduction of Ketones → LiAlH₄ / NaBH₄ → Secondary Alcohol
  • Grignard + Carbonyl → RMgX → Alcohol (حسب نوع الكربونيل)
تحضير الإيثرات
  • Dehydration of Alcohols → H₂SO₄ / 140°C → Ether
  • Williamson Synthesis → Alkoxide + 1° Alkyl Halide → Ether
⚠️ نقطة خلط شائعة: اختيار 1° alkyl halide في Williamson لتجنب elimination.

الشيت 4 – Aldehydes & Ketones

طرق التحضير
  • Oxidation of 1° Alcohol → PCC → Aldehyde
  • Oxidation of 2° Alcohol → KMnO₄ / CrO₃ → Ketone
  • Ozonolysis of Alkenes → O₃ → Aldehydes / Ketones
التفاعلات المهمة
  • Reduction → LiAlH₄ / NaBH₄ → Alcohols
  • Grignard Addition → RMgX → Alcohol (1° / 2° / 3°)
  • Aldol Condensation → Base → β-hydroxy aldehyde/ketone
⚠️ تذكير امتحاني: الألدهيد يتأكسد – الكيتون لا.

الشيت 5 – Carboxylic Acids & Derivatives

طرق تحضير الأحماض
  • Oxidation of Aldehydes → KMnO₄ / CrO₃ → Carboxylic Acid
  • Hydrolysis of Nitriles → H⁺ / OH⁻ → Carboxylic Acid
  • Grignard + CO₂ → H₃O⁺ → Carboxylic Acid (+1 C)
مشتقات الأحماض – أهم التحولات
  • Acid Chloride → SOCl₂
  • Esterification → Acid + Alcohol / H⁺ → Ester
  • Saponification → Ester + Base → Carboxylate
⚠️ ترتيب النشاط مهم في الأسئلة التحويلية.
